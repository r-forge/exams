Exam 1

  1. Question

    What does the following logo stand for?

    1. Online retailer
    2. Search engine
    3. Technology company
    4. Programming language
    5. Letter in the alphabet

    Solution

    The logo stands for the R system for statistical computing and graphics (http://www.R-project.org/).

    1. False. The R project does not sell anything.
    2. False. R is not a search engine.
    3. False. R is an open-source project, not a company.
    4. True. R is a programming language.
    5. False. Although R is (also) a letter in the alphabet, this is not what the logo represents.
