

\begin{question}

Consider the following automaton with initial state A and accepting state B:

\includegraphics[width=5cm]{automaton.pdf}

Which of the following sequences are accepted?

\begin{answerlist}
  \item 1010011
  \item 01011
  \item 00100
  \item 1101
  \item 111110
\end{answerlist}
\end{question}


\begin{solution}

The given automaton accepts input strings which consist of an
odd number of ones and
even number of zeros.

\begin{answerlist}
  \item Not accepted
  \item Accepted
  \item Accepted
  \item Not accepted
  \item Not accepted
\end{answerlist}
\end{solution}
