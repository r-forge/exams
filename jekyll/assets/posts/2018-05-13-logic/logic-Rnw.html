Exam 1

  1. Question

    Consider the following gates coding logical operators:
    A B C
    Which of these gates belongs to the following logical truth table?

    1. A
    2. B
    3. C
    4. None of these

    Solution

    The truth table codes the logical nor operator.
    The gates displayed code the following logical operators: A = and, B = nor, C = or.
    Hence, the truth table corresponds to: B.

    1. False
    2. True
    3. False
    4. False
