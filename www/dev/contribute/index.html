Contribute to Website and Project

We're looking forward to your contributions to this website and the R/exams package. This page shows you how!

Website

Typos, improvement of tutorial, more templates and everything you can think of for this website can be contributed via github. If you prefer direct contact instead of public contribution, please see our contact page.

R/exams package

The R/exams package development is hosted on R-Forge, where you can contribute via the common R infrastructure.
