Installation

R/exams Installation Tutorial for Ubuntu users

Before you can start using our delightful software package, you will need to install some other software. This tutorial will show step-by-step how to get started.

This is the first tutorial. It is part of the first step series, which tries to make beginner’s live easier.

To use R/exams you need to install:

  • LaTeX
  • R

Click on the upper links to gather information about the installation.

Afterwards you can download the whole package exams_2.2-1.tar.gz with the software and given templates.

After saving the folder you have to start R in your Terminal. Just write R and press Enter. Now type in the following code:

R> library("exams")
R> tstat_sol <- exams ("tstat.Rnw")
R> tstat_sol

If your installation was succesfull a PDF file will open.
